How it works

Three steps, no coding required

01

Upload knowledge

Drop in text, Markdown, or PDF docs through the admin dashboard.

02

Ask a question

ANSARI retrieves the relevant passages and answers with sources cited.

03

Embed anywhere

One script tag brings the widget to any page on any site.

The embed

This is the whole integration

<script src="https://your-server.com/widget.js"></script>
<script>
  ANSARI.init({
    apiKey: "pk_live_xxxxx",
    theme: "auto"
  });
</script>
